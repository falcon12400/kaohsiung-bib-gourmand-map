# 高雄必比登推介美食地圖 🗺️

2025 年高雄米其林必比登推介（Bib Gourmand）24 間餐廳互動式地圖。

## 功能

- 🗺️ 互動地圖：基於 Leaflet 開源地圖，可縮放、拖曳
- 🎨 分類標記：紅色＝台菜合菜、橘色＝小吃、綠色＝其他料理
- 🔍 搜尋＋篩選：依名稱搜尋、依類別篩選
- 📱 手機 RWD：響應式設計，手機平板都能用
- 🧭 Google Maps 導航：點擊即可開啟 Google Maps 導航

## 檔案

- `index.html` — 頁面結構
- `style.css` — 深色主題樣式
- `data.js` — 24 間餐廳資料（座標、地址、分類、描述）
- `app.js` — 地圖渲染、搜尋篩選、互動邏輯

## GitHub Pages

1. Push 到 GitHub
2. 到 repo → Settings → Pages
3. Source 選擇 `main` branch，root 選擇 `/ (root)`
4. 存取：`https://falcon12400.github.io/kaohsiung-bib-gourmand-map/`
